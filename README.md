# EEMD方法对信号进行噪声压制

## 资源简介

本仓库提供了一个用于信号噪声压制的资源文件，主要使用EEMD（Ensemble Empirical Mode Decomposition）方法对信号进行处理。该方法通过分解信号为多个本征模态函数（IMF），从而有效地去除噪声，提高信号的清晰度和可分析性。
